# ÉLAN Atelier

Elegantná prezentačná maketa webstránky pre salón krásy.

## Lokálne spustenie

```powershell
node server.mjs
```

Potom otvorte `http://localhost:4173`.

## Nasadenie

Projekt je pripravený ako statická stránka pre Vercel. Nahraďte ukážkové
fotografie, texty, služby, recenzie, adresu a telefón reálnymi údajmi salónu.
